
  # Full Website Design for DB.Learn

  This is a code bundle for Full Website Design for DB.Learn. The original project is available at https://www.figma.com/design/egUUuFVYXzf8wXNZiZOfsR/Full-Website-Design-for-DB.Learn.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run api` to start the Lab 6 API server on `http://localhost:3001`.

  Run `npm run dev` to start the Vite frontend.

  Open `/lab6` to test the SQLite + REST CRUD module.
